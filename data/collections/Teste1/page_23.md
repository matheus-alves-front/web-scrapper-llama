# Modules command-line interface (CLI) [¶](https://docs.python.org/3/library/cmdline.html\#modules-command-line-interface-cli "Link to this heading")

The following modules have a command-line interface.

- [ast](https://docs.python.org/3/library/ast.html#ast-cli)

- [asyncio](https://docs.python.org/3/library/asyncio.html#asyncio-cli)

- [`base64`](https://docs.python.org/3/library/base64.html#module-base64 "base64: RFC 4648: Base16, Base32, Base64 Data Encodings; Base85 and Ascii85")

- [calendar](https://docs.python.org/3/library/calendar.html#calendar-cli)

- [`code`](https://docs.python.org/3/library/code.html#module-code "code: Facilities to implement read-eval-print loops.")

- [compileall](https://docs.python.org/3/library/compileall.html#compileall-cli)

- [`cProfile`](https://docs.python.org/3/library/profile.html#module-cProfile "cProfile"): see [profile](https://docs.python.org/3/library/profile.html#profile-cli)

- [dis](https://docs.python.org/3/library/dis.html#dis-cli)

- [doctest](https://docs.python.org/3/library/doctest.html#doctest-cli)

- `encodings.rot_13`

- [`ensurepip`](https://docs.python.org/3/library/ensurepip.html#module-ensurepip "ensurepip: Bootstrapping the \"pip\" installer into an existing Python installation or virtual environment.")

- [`filecmp`](https://docs.python.org/3/library/filecmp.html#module-filecmp "filecmp: Compare files efficiently.")

- [`fileinput`](https://docs.python.org/3/library/fileinput.html#module-fileinput "fileinput: Loop over standard input or a list of files.")

- [`ftplib`](https://docs.python.org/3/library/ftplib.html#module-ftplib "ftplib: FTP protocol client (requires sockets).")

- [gzip](https://docs.python.org/3/library/gzip.html#gzip-cli)

- [http.server](https://docs.python.org/3/library/http.server.html#http-server-cli)

- `idlelib`

- [inspect](https://docs.python.org/3/library/inspect.html#inspect-module-cli)

- [json](https://docs.python.org/3/library/json.html#json-commandline)

- [mimetypes](https://docs.python.org/3/library/mimetypes.html#mimetypes-cli)

- [`pdb`](https://docs.python.org/3/library/pdb.html#module-pdb "pdb: The Python debugger for interactive interpreters.")

- [pickle](https://docs.python.org/3/library/pickle.html#pickle-cli)

- [pickletools](https://docs.python.org/3/library/pickletools.html#pickletools-cli)

- [platform](https://docs.python.org/3/library/platform.html#platform-cli)

- [`poplib`](https://docs.python.org/3/library/poplib.html#module-poplib "poplib: POP3 protocol client (requires sockets).")

- [profile](https://docs.python.org/3/library/profile.html#profile-cli)

- [`pstats`](https://docs.python.org/3/library/profile.html#module-pstats "pstats: Statistics object for use with the profiler.")

- [py\_compile](https://docs.python.org/3/library/py_compile.html#py-compile-cli)

- [`pyclbr`](https://docs.python.org/3/library/pyclbr.html#module-pyclbr "pyclbr: Supports information extraction for a Python module browser.")

- [`pydoc`](https://docs.python.org/3/library/pydoc.html#module-pydoc "pydoc: Documentation generator and online help system.")

- [`quopri`](https://docs.python.org/3/library/quopri.html#module-quopri "quopri: Encode and decode files using the MIME quoted-printable encoding.")

- [random](https://docs.python.org/3/library/random.html#random-cli)

- [`runpy`](https://docs.python.org/3/library/runpy.html#module-runpy "runpy: Locate and run Python modules without importing them first.")

- [site](https://docs.python.org/3/library/site.html#site-commandline)

- [sqlite3](https://docs.python.org/3/library/sqlite3.html#sqlite3-cli)

- [symtable](https://docs.python.org/3/library/symtable.html#symtable-cli)

- [sysconfig](https://docs.python.org/3/library/sysconfig.html#sysconfig-cli)

- [`tabnanny`](https://docs.python.org/3/library/tabnanny.html#module-tabnanny "tabnanny: Tool for detecting white space related problems in Python source files in a directory tree.")

- [tarfile](https://docs.python.org/3/library/tarfile.html#tarfile-commandline)

- `this`

- [timeit](https://docs.python.org/3/library/timeit.html#timeit-command-line-interface)

- [tokenize](https://docs.python.org/3/library/tokenize.html#tokenize-cli)

- [trace](https://docs.python.org/3/library/trace.html#trace-cli)

- [`turtledemo`](https://docs.python.org/3/library/turtle.html#module-turtledemo "turtledemo: A viewer for example turtle scripts")

- [unittest](https://docs.python.org/3/library/unittest.html#unittest-command-line-interface)

- [uuid](https://docs.python.org/3/library/uuid.html#uuid-cli)

- [`venv`](https://docs.python.org/3/library/venv.html#module-venv "venv: Creation of virtual environments.")

- [`webbrowser`](https://docs.python.org/3/library/webbrowser.html#module-webbrowser "webbrowser: Easy-to-use controller for web browsers.")

- [zipapp](https://docs.python.org/3/library/zipapp.html#zipapp-command-line-interface)

- [zipfile](https://docs.python.org/3/library/zipfile.html#zipfile-commandline)


See also the [Python command-line interface](https://docs.python.org/3/using/cmdline.html#using-on-general).
