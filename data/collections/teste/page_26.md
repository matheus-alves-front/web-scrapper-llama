## What [â€‹](https://ohmyposh.dev/docs/segments/cli/firebase\#what "Direct link to What")

Display the current active Firebase project.

This segment leverages the `firebase-tools.json` file generated by the [Firebase CLI](https://firebase.google.com/docs/cli).
The active files listed there are used to determine the current project against the current directory.
